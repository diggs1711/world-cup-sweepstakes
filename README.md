# World Cup 2026 Sweepstakes Dashboard 🏆

A live leaderboard for your World Cup sweepstakes. 16 players, 3 teams each, automatic scoring with underdog bonuses.

### Scoring
- Win = 3pts, Draw = 1pt
- Pot C beats Pot A = +2 bonus
- Pot C draws Pot A = +1 bonus
- Pot B beats Pot A = +1 bonus

### Deployment

[![Deploy to Netlify](https://www.netlify.com/img/deploy/button.svg)](https://app.netlify.com/start/deploy?repository=https://github.com/diggs1711/world-cup-sweepstakes)

Or deploy manually:
1. Push to GitHub
2. Connect repo in Netlify
3. Set publish directory to `/`
4. Deploy 🚀
